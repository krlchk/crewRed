Getting started
1. Install dependencies: npm install
2. Run Storybook: npm run storybook

Component Overview
1. Input Component
  A versatile input field wrapper.
  Features: Password visibility toggle, clear input button.
  Variants: Text, Password, Number.
  Storybook Path: Input/InputField.

2. Toast Notification
  Auto-dismissible popup notification located at the bottom-right corner.
  Features: Smooth slide animations, auto-dismiss timer, manual close button.
  Storybook Path: Feedback/Toast

3. Sidebar Menu
  Collapsible side navigation panel.
  Features: Slides in from the right, supports multi-level nested items, closes on overlay click.
  Storybook Path: Navigation/SidebarMenu
